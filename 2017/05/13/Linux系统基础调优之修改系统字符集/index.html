linux系统基础调优之修改系统字符集

发表于 2017-05-13 | 分类于 linux

修改系统字符集

  • 查看字符集配置文件

cat /etc/sysconfig/i18n

  • 修改字符集

  • 备份

cp /etc/sysconfig/i18n /etc/sysconfig/i18n.ori

  • 预修改

sed ‘s#en_GB#zh_CN#g’ /etc/sysconfig/i18n

  • 修改

sed -i ‘s#en_GB#zh_CN#g’ /etc/sysconfig/i18n

  • 查看

grep “LANG” /etc/sysconfig/i18n

  • 查看系统变量

echo $LANG

发现修改并未生效

  • 生效

source /etc/sysconfig/i18n

或

. /etc/sysconfig/i18n

  • 再次查看系统变量

echo $LANG

修改完成
